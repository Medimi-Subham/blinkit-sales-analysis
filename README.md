Blinkit Sales Analysis  



### Project Overview



This project analyzes Blinkit's retail sales data using SQL Server and Power BI to identify sales trends, product performance, outlet efficiency, and customer purchasing patterns.



The objective is to transform raw transactional data into actionable business insights through data cleaning, SQL analysis, and interactive dashboards.



#### Tools Used

1. SQL Server Management Studio (SSMS)
2. Power BI
3. Excel
4. GitHub





#### Dataset Features

1. Product Categories
2. Fat Content
3. Outlet Types
4. Outlet Sizes
5. Outlet Locations
6. Item Visibility
7. Product MRP
8. Outlet Establishment Year
9. Sales Revenue





#### Key Business Questions

1. Product Analysis
2. Which product categories generate the highest revenue?
3. Do Low Fat products outperform Regular products?
4. Which categories have premium pricing?
5. Outlet Analysis
6. Which outlet types generate maximum sales?
7. Which outlet size performs best?
8. Which location tier contributes most revenue?





#### Sales Insights

1. Does product visibility impact sales?
2. Is there a relationship between MRP and sales?
3. How do outlet characteristics influence revenue?





#### SQL Operations Performed

1. Data Cleaning
2. Standardization
3. Null Value Handling
4. Category Normalization



#### Dashboard Features

1. Interactive Slicers
2. KPI Cards
3. Product Analysis
4. Outlet Analysis



### Key Insights

* Fruits \& Vegetables generated the highest sales revenue.
* Supermarket Type1 contributed the largest share of total sales.
* Tier 3 locations outperformed Tier 1 and Tier 2 locations.
* Medium-sized outlets generated the highest revenue contribution.
* Low Fat products contributed a larger share of sales than Regular products.
